Ficheros

React siempre parte de un componente raíz1 en el que estarán anidados otros que completan una página concreta. Normalmente, se suelen separar los componentes en diferentes ficheros para que sea más fácil encontrarlos, mantenerlos y reusarlos en diferentes páginas o componentes.

Para compartir un componente de un fichero hay que exportarlo, y para utilizarlo desde otro componente en otro fichero hay que importarlo:

Compartido.jsPadre.js
export default function Compartido() {}import Compartido from './Compartido.js';
