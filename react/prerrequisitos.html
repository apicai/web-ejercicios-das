Prerrequisitos

Algunos ejercicios se pueden hacer directamente desde esta web gracias a Sandpack. Los navegadores recomendados son Firefox o Chrome.

Para desarrollar localmente en React será necesario:

  • El editor Visual Studio Code, que facilita el autocompletado de código
  • Node.js para montar el entorno de desarrollo. Comprueba que se instaló correctamente ejecutando en un terminal (PowerShell en Windows): 
    node -v
npm -v
