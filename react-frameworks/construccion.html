Construcción

Hasta ahora hemos utilizado React en un sandbox1. En un proyecto real, es necesario un proceso de construcción que genera los ficheros de la aplicación web optimizados2 a desplegar en un servidor web en producción.

NodeJS es el entorno de ejecución que nos permite montar el entorno de desarrollo local y construir el proyecto React. Instala los prerrequisitos y ejecuta en un terminal (PowerShell en Windows) los siguientes comandos:

  1. npm create vite@latest mi-app-react -- --template react
cd mi-app-react
npm install
npm run dev
  2. Teclea o ↲ en el terminal
  3. Abre el directorio mi-app-react con el VSCode, y modifica el JSX del fichero src/App.jsx, guarda y observa el resultado en el navegador
  4. Vuelve al terminal y teclea q ↲, y ejecuta: 
    npm run build
npm run preview
  5. Vuelve a repetir los pasos 2 y 3

La optimización consiste en "transpilar" el código para hacerlo compatible con la versión de JS soportada por la mayoría de navegadores, y en paquetizar todas las dependencias y el código del proyecto (eliminando el código no utilizado, "minificando" el código para reducir su tamaño, y dividiéndolo en módulos independientes para cargar sólo los necesarios al entrar)
