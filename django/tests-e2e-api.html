Se pueden realizar tests automáticos con el LiveServerTestCase de Django, que permite hacer llamadas HTTP reales al servidor, ya que dicha clase lo levanta al empezar y lo para al finalizar.

Se pueden hacer tests manuales utilizando herramientas externas de testing como Postman, Httpie o REST Client. Con ellas, también se pueden probar servidores desplegados en otras máquinas (no necesariamente ejecutándose en local).

En el caso de REST Client podemos probar definiendo las peticiones en un fichero de texto:

### Creación

POST http://127.0.0.1:8000/api/recurso/
Content-Type: application/json

{
  "campo": "petición"
}

### Recuperación

GET http://127.0.0.1:8000/api/recurso/1
Cabecera: valor
